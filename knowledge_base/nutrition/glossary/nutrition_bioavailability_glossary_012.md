# Биодоступность

## Определение
Доля нутриента, поступившего с пищей, которая усваивается, попадает в системный кровоток и может быть использована организмом для физиологических функций. Зависит от формы нутриента, состава пищи, состояния ЖКТ и индивидуальных особенностей.

## Синонимы
- Усвояемость
- Доступность для усвоения
- Bioavailability (англ.)

## Контекст использования
Термин «биодоступность» используется в нутрициологии, диетологии, гастроэнтерологии и пищевых технологиях для объяснения того, почему количество нутриента в продукте (химический состав) не равно его реальной пользе для организма. 

## Примеры употребления
- «Биодоступность негемового железа из растительных продуктов составляет 2–20% и повышается при наличии витамина C».
- «Каротиноиды из моркови лучше усваиваются при употреблении с жирами, что увеличивает их биодоступность».
- «Фитиновая кислота в бобовых и злаках может снижать биодоступность цинка и железа».
