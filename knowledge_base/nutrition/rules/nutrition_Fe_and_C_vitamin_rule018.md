# Сочетать источники железа с витамином C для улучшения усвоения

## Суть рекомендации
Сочетать продукты, богатые негемовым железом (растительные источники: бобовые, шпинат, киноа), с источниками витамина C (цитрусовые, болгарский перец, брокколи) для повышения биодоступности железа на 2–4 раза.

## Условия применимости
- **Целевая группа**: вегетарианцы, веганы, беременные, лица с дефицитом железа или риском анемии
- **Контекст**: рацион с преобладанием растительных источников железа
- **Предварительные условия**: подтверждённый дефицит железа или повышенная потребность в нём

## Обоснование
Железо в растительных продуктах представлено в негемовой форме, усвоение которой составляет 2–20% и сильно зависит от состава приёма пищи. Витамин C восстанавливает трёхвалентное железо (Fe³⁺) в двухвалентное (Fe²⁺), которое лучше всасывается в кишечнике, и образует с ним растворимые комплексы. Это особенно важно для групп с повышенными потребностями или ограниченным потреблением гемового железа.

## Исключения и противопоказания
- Гемохроматоз: ограничение потребления железа и витамина C по назначению врача
- Приём препаратов железа: витамин C может усиливать всасывание, но требует контроля дозы
- Заболевания ЖКТ: при мальабсорбции стратегия может быть недостаточно эффективной

## Практическое применение
1. Добавляйте лимонный сок к салатам со шпинатом, чечевицей или киноа
2. Сочетайте бобовые с овощами, богатыми витамином C: перец, томаты, брокколи
3. Употребляйте фрукты с высоким содержанием витамина C (киви, цитрусовые, ягоды) как десерт после основного приёма пищи с железом
4. Избегайте чая, кофе, кальция в один приём с источниками железа — они снижают усвоение
5. При приготовлении: добавляйте кислоту (уксус, томатную пасту) к блюдам с бобовыми и зеленью

## Уровень доказательности
**High** — биохимические и клинические исследования подтверждают механизм и эффективность сочетания витамина C с негемовым железом для улучшения его усвоения.
