# Индекс массы тела

## Определение
Индекс массы тела (ИМТ) — это показатель, рассчитываемый как отношение массы тела к квадрату роста, используемый для оценки избыточного веса или ожирения. Он помогает определить риски для здоровья.

## Синонимы
- ИМТ
- BMI

## Контекст использования
Используется в разделах "Здоровье" и "Фитнес" ассистента ЗОЖ, связан с термином сжигание жира.

## Примеры употребления
- "Индекс массы тела помогает отслеживать прогресс в похудении."
