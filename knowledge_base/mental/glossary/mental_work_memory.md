# Рабочая память
## Определение
Рабочая память — это когнитивная система с ограниченной ёмкостью, отвечающая за временное хранение и манипуляцию информацией, необходимой для выполнения сложных задач: понимания речи, решения проблем, принятия решений и обучения.

## Синонимы
- Оперативная память (в когнитивном контексте)
- Кратковременная рабочая память
- Working memory (англ.)

## Контекст использования
Центральное понятие когнитивной психологии и нейронаук. Применяется в разделах «Обучение и продуктивность», «Внимание и концентрация», «Когнитивное старение». Модель Бэддели выделяет четыре компонента: фонологическую петлю (вербальная информация), визуально-пространственный блок, центральный исполнитель (управление вниманием) и эпизодический буфер. Ёмкость составляет около 4±1 «единиц» информации без внешних опор.

## Примеры употребления
- «Рабочая память критична для многозадачности: при перегрузке этой системы возрастает количество ошибок и время реакции».
- «Стратегии когнитивной разгрузки (списки, заметки) компенсируют ограниченную ёмкость рабочей памяти, повышая эффективность решения задач».
